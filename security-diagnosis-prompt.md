# セキュリティ脆弱性診断プロンプト v2.0
# （改訂版 + デザイン改善 + 補足情報反映）

このプロンプトをコピーして、Claude.aiの会話に貼り付けて使います。
`（　）`の中だけ書き換えてください。

---

## 使い方

1. 下の「プロンプト本文」をすべてコピーする
2. 4箇所を書き換える（URL・所有区分・補足メモ・サイトの特徴）
3. Claude.aiの会話に貼り付けて送信する

---

Deep research の実行方針
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

レポート作成前に、必ず以下の観点で調査してください。

1. 対象サイト固有の調査
   - 対象URLの公開HTML・title・meta description・generator metaタグ
   - コメント内の内部情報・エラーメッセージ
   - メールアドレス・電話番号・住所等の公開情報
   - APIキー・トークン・環境変数らしき文字列の露出
   - 外部script・link・iframe
   - GTM・GA4・Clarity・Meta Pixel等の分析タグ（コンテナIDを特定）
   - Auth0・Firebase・Cognito等の認証基盤痕跡
   - Stripe・PAY.JP・GMO・Square等の決済スクリプト痕跡
   - WordPress・Next.js・React・Vue・Nuxt・Shopify・Contentful等の技術痕跡
   - 問い合わせフォーム・採用ページ・資料請求・ログイン・決済の有無

2. HTTPヘッダー調査
   可能であれば以下を確認してください。
   curl -sSI -L https://対象URL/

   確認するヘッダー：
   - Server・X-Powered-By
   - Content-Security-Policy
   - Strict-Transport-Security
   - X-Frame-Options
   - X-Content-Type-Options
   - Referrer-Policy
   - Permissions-Policy
   - Set-Cookie（Secure・HttpOnly・SameSite属性）
   - Cache-Control
   - Cross-Origin-Opener-Policy
   - Cross-Origin-Resource-Policy
   - Cross-Origin-Embedder-Policy

   取得できなかった場合は必ず「未確認」と書き、推測で補完しないこと。

3. 技術スタック別のDeep research
   確認または推定された技術に応じて、必ず最新情報を調査してください。

   Next.jsが確認または推定された場合
   →「Next.js CVE 最新」「CVE-2025-29927 Next.js」「Next.js middleware authorization bypass」
   → NVD・Vercel公式アドバイザリを確認

   GTMが確認された場合
   → コンテナIDを特定してHTMLに明示
   → 「GTMサプライチェーン攻撃」「Magecart GTM」を検索

   Contentful / Headless CMSが確認された場合
   → Space IDを特定・Delivery API Keyとの区別を確認
   → 「Contentful APIキー 露出 漏洩 事例」を検索

   Vercel / Netlify / Cloudflare / AWSが確認された場合
   → 「Vercel セキュリティ 脆弱性 最新」を検索
   → CDN/WAF設定・セキュリティヘッダーを確認

   WordPressが確認された場合
   → WordPress本体バージョン・テーマ名・プラグイン名を確認
   → xmlrpc.php・wp-json露出を確認
   → 「WordPress CVE 最新」を検索

   Auth0・Firebase・Cognitoが確認された場合
   → 「[認証サービス名] 設定不備 脆弱性 最新」を検索
   → OAuthリダイレクトURI設定・トークン漏洩事例を確認

   決済スクリプトが確認された場合
   → 「Eスキマー 決済 最新 国内」を検索
   → PCI DSS観点でのリスクを評価

   技術スタックが特定できない場合
   → 「[ドメイン名] セキュリティ インシデント」を検索
   → URLから推測できるカテゴリで検索

   補足情報から追加調査が必要な場合
   → 【補足メモ】【サイト・ビジネスの特徴】の内容から
     関連する最新のインシデント・CVE・統計を追加検索すること

4. 参照すべき公的・準公的・一次情報
   - OWASP Top 10（最新版）
   - NVD / CVE公式情報
   - JPCERT/CC 注意喚起
   - IPA 情報セキュリティ10大脅威 最新版
   - 警察庁 サイバー犯罪関連統計
   - フィッシング対策協議会 月次・年次報告
   - IBM Cost of a Data Breach 最新版
   - Verizon DBIR 最新版
   - CISA Advisory
   - ベンダー公式セキュリティアドバイザリ
   - 企業の公式インシデント報告
   - 規制当局・監督機関の公表資料
