# セキュリティ脆弱性診断【構成C】セットアップガイド

## 必要なファイル（同じフォルダに置く）

```
📁 security-diagnosis/
  ├── input.html                    ← INPUT画面（ブラウザで開く）
  ├── server.js                     ← Node.jsサーバー
  └── security-diagnosis-prompt.md ← プロンプト（既存）
```

---

## 起動手順

### ① Claude Codeアプリでフォルダを開く

上記のフォルダをClaude Codeアプリで開きます。

### ② Claude Codeに指示する

コマンド欄に以下を入力します。

```
server.jsを起動して
```

以下のメッセージが表示されれば起動完了です。

```
セキュリティ脆弱性診断サーバー 起動完了
ポート: 3001
```

### ③ input.htmlをブラウザで開く

`input.html` をダブルクリック、またはブラウザで開きます。

### ④ 診断を実行する

1. 動作モードを選択（テスト / 本番）
2. 使用モデルを選択
3. 診断対象のURLを入力
4. 自社 / 他社を選択
5. 補足メモをプルダウンで選択（任意）
6. 「レポートを生成する」ボタンを押す

### ⑤ レポートを確認・保存する

レポートが同じ画面に表示されます。
「PDFで保存」ボタンを押すと、ブラウザの印刷画面が開きます。
印刷先を「PDFに保存」に設定してダウンロードしてください。

---

## 注意事項

- `server.js` の起動中はClaude Codeアプリを閉じないでください。
- ポート3001が使用中の場合は、Claude Codeに「PORT=3002でserver.jsを起動して」と指示してください。その場合、`input.html` 内の `localhost:3001` を `localhost:3002` に変更してください。
- APIキーはClaude Codeの認証情報を使用します。別途取得・設定は不要です。
